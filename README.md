## ✨ Key Features

* **💎 Premium Modern UI:** Designed with a custom purple gradient theme (`#5D3FD3` & `#8B5CF6`), glassmorphism elements, and smooth hover animations.
* **🌓 Dark & Light Mode:** Fully integrated theme toggle that syncs with user system preferences or allows manual switching.
* **📱 100% Responsive:** Flawless viewing experience across mobile, tablet, and desktop devices.
* **⚡ Highly Optimized:** Built using Tailwind CSS via CDN and vanilla JavaScript for maximum speed without heavy dependencies.
* **📬 Functional Contact Form:** Integrated with FormSubmit to send messages directly to email without requiring a backend server.
* **💼 Project Showcase:** Beautifully structured cards to display recent works, tech stacks, and upcoming plans.

---

## 🛠️ Technologies Used

* **Frontend:** HTML5, CSS3, Tailwind CSS
* **Scripting:** Vanilla JavaScript (DOM manipulation, Theme toggling)
* **Typography:** Montserrat (Google Fonts)
* **Icons:** Font Awesome 6.4.0
* **Form Handling:** FormSubmit API
