# EVS Technology - Personal Portfolio 🚀

Welcome to the **EVS Technology** portfolio repository! This is the personal portfolio website of **EV Sanjeevan**, a self-taught Full-Stack Developer passionate about building robust, scalable, and intuitive web applications. 

🔗 **Live Demo:** [EVS Technology Portfolio](https://evanjeevan07.github.io/Myportolio/)

## 🌟 Overview

This portfolio is a fully responsive, modern single-page web application (SPA) designed to showcase my journey from Botany to Binary. It features a seamless dark/light mode toggle, smooth scrolling, and custom CSS animations, all wrapped in an elegant purple-themed UI.

## ✨ Key Features

* **📱 Fully Responsive:** Optimized for all devices (Mobile, Tablet, Desktop).
* **🌓 Dark & Light Mode:** Integrated theme toggling feature with smooth transitions and system preference detection.
* **🎨 Modern UI/UX:** Built with Tailwind CSS, featuring glassmorphism effects, gradient text, and floating blur backgrounds.
* **🚀 Smooth Animations:** Custom CSS keyframes for animated logos, shimmer effects, and interactive hover states.
* **📧 Functional Contact Form:** Integrated with FormSubmit for direct email communication.
* **📄 Resume Integration:** Quick access button to download the latest professional resume.

## 🛠️ Tech Stack

* **Structure:** HTML5
* **Styling:** Tailwind CSS (via CDN) & Custom CSS
* **Interactivity:** Vanilla JavaScript
* **Icons:** FontAwesome (v6.4.0)
* **Typography:** Google Fonts ('Montserrat')
