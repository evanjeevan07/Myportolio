# EVS Technology - Full-Stack Developer Portfolio 🚀

**EVS Technology** is the personal portfolio website of **EV Sanjeevan**.  
It is designed as a modern, fast, and fully responsive website to showcase my skills, projects, and experience to the world.

---

## ✨ Features

- **Premium Design:** Modern UI with Montserrat font and a stylish purple theme  
- **Dark Mode:** Supports both system preference and manual toggle  
- **Responsive:** Works perfectly on mobile, tablet, and desktop devices  
- **Animated UI:** Smooth animations and engaging hover effects  
- **Pricing Plans:** Well-structured pricing section for clients  
- **Contact Form:** Easy communication via email and social media  

---

## 🛠️ Tech Stack

- **Frontend:** HTML5, Tailwind CSS  
- **Fonts:** Montserrat (Google Fonts)  
- **Icons:** Font Awesome 6.4.0  
- **Scripting:** JavaScript (Theme toggle & Mobile menu)  
- **Design Style:** Glassmorphism & Modern Gradient UI
