<h2 style="color: #5D3FD3;">🌟 Overview</h2>

<p style="font-size: 15px; line-height: 1.6; color: #4b5563;">
  This portfolio is a fully responsive, modern single-page web application (SPA) designed to showcase my journey from Botany to Binary. It features a seamless dark/light mode toggle, smooth scrolling, and custom CSS animations, all wrapped in an elegant purple-themed UI.
</p>

<h2 style="color: #5D3FD3;">✨ Key Features</h2>

<p style="font-size: 15px; line-height: 1.6; color: #4b5563;">
  <ul>
    <li><strong>📱 Fully Responsive:</strong> Optimized for all devices (Mobile, Tablet, Desktop).</li>
    <li><strong>🌓 Dark & Light Mode:</strong> Integrated theme toggling feature with smooth transitions and system preference detection.</li>
    <li><strong>🎨 Modern UI/UX:</strong> Built with Tailwind CSS, featuring glassmorphism effects, gradient text, and floating blur backgrounds.</li>
    <li><strong>🚀 Smooth Animations:</strong> Custom CSS keyframes for animated logos, shimmer effects, and interactive hover states.</li>
    <li><strong>📧 Functional Contact Form:</strong> Integrated with FormSubmit for direct email communication.</li>
    <li><strong>📄 Resume Integration:</strong> Quick access button to download the latest professional resume.</li>
  </ul>
</p>

<h2 style="color: #5D3FD3;">💻 Getting Started (Local Setup)</h2>

<p style="font-size: 15px; line-height: 1.6; color: #4b5563;">
  Since this is a static website, setting it up locally is extremely simple. No build tools or package managers are required.
</p>

<pre><code>git clone https://github.com/evanjeevan07/Myportolio.git
cd Myportolio
</code></pre>

<p style="font-size: 15px; line-height: 1.6; color: #4b5563;">
